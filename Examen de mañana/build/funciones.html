funciones module¶

funciones.cuadrados(lista)[fuente]¶

Funcion que hace el cuadrado de los elementos de una lista :param lista: lista de numeros :return: lista de cuadrados

>>> lista = [1,2,3,4]
>>> cuadrados(lista)
[1, 4, 9, 16]
funciones.nombre_de_la_funcion(parametro1, parametro2)[fuente]¶

Funcion de ejemplo para mostar como se escriben las funciones.
